# AMAZON DISCOUNT STARTEGY ANALYSIS

## OBJECTIVE
To design a data-driven discount strategy that maximizes revenue while maintaining strong customer satisfaction, by analyzing pricing patterns, customer engagement, and product performance across 1,000+ Amazon listings.

## TOOLS USED
1. Python (Pandas, Matplotlib) – Data cleaning, feature engineering, and exploratory analysis
2. Power BI – Interactive dashboard and visualization
3. Excel – Data validation and preprocessing

## KEY INSIGHT
1. Identified an optimal 20–30% discount range that maximizes revenue efficiency.
2. Discounts beyond 50% lead to diminishing returns, increasing engagement but not contributing proportionally to revenue.
3. Product ratings remain consistently high (~4.0+) across all discount levels, suggesting that heavy discounting does not significantly improve perceived value.
4. Higher discounts attract more user interaction (ratings/reviews), but do not translate into proportional revenue gains.
5. Revenue is concentrated in a few key categories, while some categories show inefficient discounting (high discount, low return).

## FILES
1. Dataset – 1,000+ Amazon product records.
2. Python Analysis – Data preprocessing, feature engineering, and EDA.
3. Power BI Dashboard – Interactive visualization with dynamic filtering.

## OUTCOME
Delivered actionable pricing strategy recommendations, enabling businesses to:

1. Maximize revenue through optimized discounting
2. Maintain high customer satisfaction (avg rating ~4.0+)
3. Avoid inefficient high-discount strategies

## BUSINESS IMPACT
This project demonstrates how analytical insights can support strategic pricing decisions, helping e-commerce platforms improve profitability while sustaining customer trust.
